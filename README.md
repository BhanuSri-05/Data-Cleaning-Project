Data Cleaning Project
Project Overview

This project focuses on cleaning and preparing the Superstore dataset for further analysis. Data cleaning is an essential step in the data analytics process as it improves data quality and ensures accurate insights.

Objective

The main objective of this project is to:

Identify and handle missing values
Remove duplicate records
Verify data consistency
Convert data into appropriate formats
Create a clean dataset ready for analysis
Dataset

Dataset Name: Sample Superstore Dataset

The dataset contains information about:

Orders
Customers
Products
Sales
Profit
Shipping details

Data Cleaning Steps Performed
1. Load Dataset
Imported the dataset using Pandas.
Displayed the first few rows for inspection.
2. Dataset Inspection
Checked dataset dimensions.
Reviewed column names and data types.
3. Missing Value Analysis
Identified missing values in all columns.
Verified data completeness.
4. Duplicate Record Check
Detected duplicate rows.
Removed duplicates where necessary.
5. Data Type Validation
Ensured columns had appropriate data types.
Converted date columns to datetime format if required.
6. Final Verification
Checked the cleaned dataset.
Confirmed data quality improvements.
